# South West Cornwall Group — Riding for the Disabled

Website source for [rdasouthwestgroup.org.uk](https://rdasouthwestgroup.org.uk).

The South West Cornwall Group is a member group of the [Riding for the Disabled Association](https://www.rda.org.uk), providing therapeutic horse riding and equine activities for people with disabilities and health challenges since 1976. Based at Old Mill Stables, Lelant Downs, Cornwall.

**Charity No. 1073713**

## Hosting
